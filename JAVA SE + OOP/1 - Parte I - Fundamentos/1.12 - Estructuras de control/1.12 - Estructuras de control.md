## 1) Descripción  

Las **estructuras de control** permiten decidir **qué código se ejecuta, cuándo y cuántas veces**.  
Son fundamentales porque hacen que un programa deje de ser lineal y pase a ser **dinámico y lógico**.  
  
En este tema vas a aprender a:  
- Tomar decisiones (`if`, `else`)  
- Evaluar múltiples casos (`switch`)  
- Repetir acciones (`while`, `for`, `do-while`)  
  
Esto es clave para cualquier sistema, especialmente para tu proyecto RPG (combate, turnos, eventos, etc.).  


---  
  
## 2) Teoría  
  
###  1. Estructuras condicionales (decisiones)  
  
###  IF  

Ejecuta un bloque **si se cumple una condición**.

```java
if (vida > 0) {  
System.out.println("El personaje sigue vivo");  
}
```

### IF - ELSE  

Permite elegir entre dos caminos.

```java
if (vida > 0) {  
System.out.println("Sigue vivo");  
} else {  
System.out.println("Está muerto");  
}
```

### IF - ELSE IF - ELSE  

Para múltiples condiciones.

```java
if (vida > 70) {  
System.out.println("Salud alta");  
} else if (vida > 30) {  
System.out.println("Salud media");  
} else {  
System.out.println("Salud baja");  
}
```

---  
  
###  2. SWITCH (múltiples casos)  
  
Se usa cuando comparás **una variable contra varios valores posibles**.

```java
int opcion = 2;

switch (opcion) {  
case 1:  
System.out.println("Atacar");  
break;  
case 2:  
System.out.println("Defender");  
break;  
case 3:  
System.out.println("Huir");  
break;  
default:  
System.out.println("Opción inválida");  
}
```

---  
  
###  3. Bucles (repetición)  
  
## WHILE  

Se ejecuta **mientras la condición sea verdadera**.

```java
int vida = 100;

while (vida > 0) {  
System.out.println("El personaje sigue luchando");  
vida -= 20;  
}
```

---  
  
### DO-WHILE 
Se ejecuta **al menos una vez**, aunque la condición sea falsa.

```java
int energia = 0;

do {  
System.out.println("Intentando acción...");  
} while (energia > 0);
```

---  
  
## FOR  

Ideal cuando sabés **cuántas veces repetir**.

```java
for (int i = 1; i <= 5; i++) {  
System.out.println("Turno: " + i);  
}
```

---  
  
## Conceptos clave  
  
- Condición → siempre devuelve `true` o `false`  
- Bloque → código dentro de `{ }`  
- Iteración → cada repetición de un bucle  
- Control de flujo → cómo se mueve el programa  
  
---  
  
  
## 3) Integración RPG  
  
Las estructuras de control son el **corazón del juego**:  
  
- `if` → decidir si el personaje vive o muere  
- `switch` → elegir acciones del jugador  
- `while` → mantener el combate activo  
- `for` → controlar turnos  
  
Ejemplo conceptual:

```java
while (vidaEnemigo > 0 && vidaJugador > 0) {  
// turno de combate  
}
```

---  
  
## 4) Notas adicionales  
  
⚠️ Errores comunes:  
- Olvidar `break` en `switch`  
- Condiciones mal planteadas (`=` en vez de `==`)  
- Bucles infinitos (`while(true)` sin salida)  
  
✅ Buenas prácticas:  
- Usar nombres claros (`vidaJugador`, no `v`)  
- Evitar lógica demasiado compleja en un solo `if`  
- Identar correctamente el código
