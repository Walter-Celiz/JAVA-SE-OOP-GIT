### Nivel 1: Entrada básica  

- Pedir nombre del jugador  
- Pedir edad  
- Mostrar mensaje personalizado  
  
### Nivel 2: Menú simple RPG 

Crear un menú con opciones:  
1. Atacar  
2. Defender  
3. Huir  
  
El programa debe responder según la opción elegida.  
  
### Nivel 3: Combate con decisiones  

Modificar el combate:  
- Si elige atacar → daño al enemigo  
- Si elige defender → reduce daño recibido  
- Si elige huir → termina el combate  
  
### Nivel 4: Validación de entrada

Evitar que el usuario ingrese opciones inválidas:  
- Si el número no es 1, 2 o 3 → volver a pedir  
  
### Nivel 5: Nombre del personaje  

- Pedir nombre del jugador  
- Mostrarlo en cada turno  
Ejemplo: `Turno de Walter`
  
### Nivel 6: Sistema de pociones 

Agregar opción:  
4. Usar poción  
- Recupera vida  
- Solo puede usarse cierta cantidad de veces
