## 1) Descripción

En este tema aprenderás a utilizar **Arrays (Arreglos)**, una estructura de datos que permite almacenar múltiples valores del mismo tipo dentro de una sola variable.

Esto es fundamental porque:

- Permite organizar grandes cantidades de datos
    
- Reduce la necesidad de crear muchas variables similares
    
- Facilita el recorrido y procesamiento de información mediante bucles
    
- Es la base para estructuras más avanzadas como matrices y colecciones
    
- Será muy utilizado en el desarrollo del RPG
    

---

## 2) Teoría

### ¿Qué es un Array?

Un Array es una estructura que almacena varios valores del mismo tipo en posiciones consecutivas de memoria.

Sin arrays:

```
int enemigo1 = 100;
int enemigo2 = 120;
int enemigo3 = 80;
```

Con arrays:

```
int[] enemigos = {100, 120, 80};
```

Ahora todos los valores están agrupados dentro de una única variable.

---

### Declaración de un Array

La declaración indica qué tipo de dato almacenará el array.

```java
int[] numeros;
```

También puede escribirse así:

```java
int numeros[];
```

Sin embargo, la forma recomendada es:

```java
int[] numeros;
```

---

### Creación de un Array

Para crear un array se utiliza la palabra reservada `new`.

```java
int[] numeros = new int[5];
```

Esto crea un array capaz de almacenar 5 números enteros.

Visualmente:

```
Índice:  0   1   2   3   4
Valor:   0   0   0   0   0
```

Los elementos se inicializan automáticamente con valores por defecto.

---

### Inicialización Directa

También es posible crear e inicializar un array en una sola línea.

```java
int[] numeros = {10, 20, 30, 40, 50};
```

Java calcula automáticamente el tamaño del array.

---

### Índices

Cada elemento posee una posición llamada índice.

Importante:

- El primer índice siempre es 0
    
- El segundo índice es 1
    
- El tercero índice es 2
    

Ejemplo:

```java
int[] numeros = {10, 20, 30, 40, 50};
```

Representación:

```
Índice:  0    1    2    3    4
Valor:  10   20   30   40   50
```

---

### Acceder a Elementos

Para acceder a un valor se utiliza su índice.

```java
System.out.println(numeros[0]);
```

Salida:

```
10
```

Otro ejemplo:

```java
System.out.println(numeros[2]);
```

Salida:

```
30
```

---

### Modificar Elementos

Los elementos pueden modificarse utilizando su índice.

```java
numeros[2] = 100;
```

Resultado:

```
Índice:  0    1    2    3    4
Valor:  10   20  100   40   50
```

---

### Tamaño del Array

Todos los arrays poseen la propiedad `length`.

```java
System.out.println(numeros.length);
```

Salida:

```
5
```

Importante:

```java
numeros.length
```

Correcto.

```java
numeros.length()
```

Incorrecto.

---

### Recorrido con for

La forma más común de recorrer un array es mediante un bucle `for`.

```java
for (int i = 0; i < numeros.length; i++) {
    System.out.println(numeros[i]);
}
```

Esto permite acceder a cada posición utilizando el índice.

---

### Recorrido con foreach

Java proporciona una sintaxis simplificada para recorrer arrays.

```java
for (int numero : numeros) {
    System.out.println(numero);
}
```

Ventajas:

- Más legible
    
- Menos errores
    
- Ideal cuando no se necesita el índice
    

---

### Valores por Defecto

Cuando un array se crea utilizando `new`, Java asigna automáticamente valores iniciales.

|Tipo|Valor por defecto|
|---|---|
|int|0|
|double|0.0|
|boolean|false|
|char|'\u0000'|
|String|null|

Ejemplo:

```java
int[] numeros = new int[3];
```

Contenido inicial:

```
[0, 0, 0]
```

---

### Error Común: Índice Fuera de Rango

Supongamos:

```java
int[] numeros = {10, 20, 30};
```

Si intentamos acceder a:

```java
numeros[3]
```

Java producirá:

```java
ArrayIndexOutOfBoundsException
```

Porque los índices válidos son:

```
0
1
2
```

## 4) Notas adicionales

### Buenas prácticas

- Utilizar nombres descriptivos
    
- Recorrer arrays utilizando `length`
    
- Utilizar `foreach` cuando no se necesite el índice
    
- Mantener todos los elementos del mismo tipo
    

---

### Errores comunes

- Comenzar desde índice 1 en lugar de 0
    
- Acceder a posiciones inexistentes
    
- Confundir `length` con `length()`
    
- Recorrer utilizando números fijos
    

Incorrecto:

```java
for (int i = 0; i < 5; i++)
```

Correcto:

```java
for (int i = 0; i < numeros.length; i++)
```

---

### Tip profesional

Cuando necesites procesar todos los elementos de un array, utiliza siempre su propiedad `length`.

```java
for (int i = 0; i < numeros.length; i++) {
    System.out.println(numeros[i]);
}
```

Esto hace que el código sea más flexible y evita errores si el tamaño del array cambia.

---

## Resumen

Los Arrays permiten almacenar múltiples valores del mismo tipo dentro de una única variable.

Cada elemento posee una posición llamada índice, que siempre comienza en 0.

Los arrays pueden recorrerse utilizando bucles `for` o `foreach`, permiten acceder y modificar elementos mediante índices y poseen una propiedad llamada `length` que indica su tamaño.

Dominar los arrays es fundamental para trabajar con inventarios, estadísticas, enemigos y muchos otros sistemas que utilizaremos durante el desarrollo del RPG.
