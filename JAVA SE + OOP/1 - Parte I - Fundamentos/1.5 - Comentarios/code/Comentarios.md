## 1) Código

Ejemplo práctico ubicado en la carpeta `code/`:

**Archivo:** `Comentarios.java`

```java
public class Comentarios {

    public static void main(String[] args) {

        // Comentario de una sola línea
        System.out.println("Hola Java");

        /*
         Comentario de múltiples líneas
         usado para explicar un bloque de código
        */
        int numero = 10;

        /**
         * Comentario de documentación
         * normalmente usado en métodos y clases
         */
        System.out.println("Número: " + numero);
    }
}
```

---
## 2) Ejercicios

1. Agrega comentarios de una línea explicando cada instrucción del `main`.
2. Usa un comentario multilínea para describir qué hace el programa completo.
3. Crea un comentario Javadoc sobre la clase `Comentarios`.
4. Elimina comentarios innecesarios y deja solo los útiles.
